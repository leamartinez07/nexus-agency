# Nexus Agency

> Portfolio project by [Leandro Martinez](https://leandromartinez.vercel.app) · [GitHub](https://github.com/leamartinez07)

Custom landing page built with WordPress and PHP. Developed as a portfolio project to demonstrate front-end skills within the WordPress ecosystem.

## Features

- Fixed navigation bar with smooth scroll to sections
- Hero section with call-to-action
- Animated timeline for services or process steps
- FAQ section with accordion expand/collapse
- Contact form
- Fully responsive design (mobile, tablet, desktop)
- WCAG 2.1 accessibility standards applied
- SEO-optimized structure

## Tech Stack

| Layer | Technology |
|---|---|
| CMS | WordPress |
| Template | PHP (custom page template) |
| Styling | CSS3 |
| Scripting | JavaScript |
| Base Theme | Twenty Twenty-Five |

## Files

- `template-nexus.php` — WordPress custom page template. Assign it to any page from the Page Attributes panel in the WordPress editor.
- `nexus-agency.html` — Static HTML export of the landing page. Can be opened directly in a browser without WordPress.

## Local Setup (WordPress)

1. Copy `template-nexus.php` into your active theme folder (e.g. `wp-content/themes/your-theme/`)
2. In the WordPress admin, create or edit a page
3. In the Page Attributes panel, select the "Nexus Agency" template
4. Publish the page

## Preview (Static)

Open `nexus-agency.html` directly in a browser for a quick preview without needing a WordPress installation.
